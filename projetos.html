Projeto Livraria Virtual

2018
HTML css JavaScript TypeScript sass React Next JS Bulma Responsivo Banco de Dados MongoDB
GitHub Ir para o site

O Livraria Virtual foi inicialmente um projeto acadêmico de conclusão de período. Seu objetivo era ser uma aplicação web com funções de persistir e recuperar informações em banco de dados SQL. Criado em 2018, codificado em java, e construído com maven em sua versão original. Esta é uma recriação melhorada através de tecnologias como Next JS, React, a biblioteca css Bulma, e a plaforma de dados MongoDB.

Por sua natureza de aplicação web, com uma interface responsiva pode ser acessada via computador desktop ou dispositivo mobile.

A aplicação foi integrada ao banco de dados da plataforma MongoDB, para a realização das rotinas de persistência de dados, como o cadastro, e a recuperação dos mesmos.

A Página Home realiza uma consulta aos dados para recuperar os livros cadastrados em forma de produtos que mais interessaram os usuários.

LabVad Unigranrio

2019
C# .Net HTML JavaScript Bootstrap JQuery css MVC Responsivo Azure
GitHub Ir para o site

O LabVad Unigranrio foi um projeto de conclusão de curso em conjunto com a universidade, e em parceria com o LabVad UFRJ. Sua proposta era de ser uma ferramenta para aprendizado de programação através de experimentos práticos em Arduino, e incentivo à área de robótica.

Através da plataforma, o aluno teria acesso a diferentes experimentos no laboratório virtual. Poderia programar e ver seu código sendo executado em tempo real, em componentes Arduino alocados na Unigranrio.

O projeto foi descontinuado antes de sua efetivação.

O cadastro não está disponível para essa versão de demonstração.

Na aba de Agendamento, o usuário tem a sua disposição um calendário para o agendamento de dia e hora em que pretende usar dos componentes. O calendário foi feito por meio da bilbioteca em javascript, FullCalendar.

Após a escolha do dia, o usuário é direcionado para a seleção da hora, e do tempo de uso pretendido, em uma margem de 30min a 2h.

Na aba de Experimento, uma vez realizado o agendamento, o usuário tem acesso a um painel com editor de código, onde é possível programar para Arduino. Ao executar seu código, a caixa de componentes Arduino poderia ser vista executando ao vivo as instruções programadas pelo usuário. Para o editor de código foi utilizado a biblioteca Ace, em javascript.

O painel também permite acessar códigos salvos, ou gravações anteriores de uso dos componentes.

Também está disponível uma funcionalidade para alterar o esquema de cores da plaforma, no menu principal.

A execução de código e transmissão ao vivo da caixa de componentes não estão disponíveis para esta versão de demonstração.

A aba de Minha Conta permite o gerenciamento básico dos dados e da conta do usuário cadastrado na plataforma. Há uma seção reservada para ver agendamentos realizados para o futuro.

A aba de Ajuda reune as principais informações de cada componente. O que são, como funcionam, o esquema de conexão entre eles e um circuito Arduino, além de um exemplo de código que serve de referência para programar instruções a um componente.

WDesign

2022
HTML css JavaScript Bootstrap sass Responsivo
GitHub Ir para o site

Um exemplo de Landing Page para portfólio de Web Design. Interface moderna, limpa e totalmente responsiva. Uma apresentação elegante animada com JavaScript, e espaço para textos de apresentação e oferta de serviços.

A seção de Projetos é uma galeria de imagens com trabalhos de design gráfico. Cada imagem tem vinculada um popover reservado para uma descrição única do projeto, permitindo que informações importantes do trabalho possam ser destacadas. O popover é ativado ao interagir com a respectiva imagem.

A seção de Contato possui uma área dedicada para links de redes sociais e um serviço de envio de mensagem diretamente para um correio eletrônico profissional. O rodapé do site possui um apanhado elementar das principais informações reunidas do web designer.
